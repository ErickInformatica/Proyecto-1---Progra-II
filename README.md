# Proyecto 1 - Programacion II - UMG
## Sistema de Tickets
Este proyecto esta orientado a poder manegar Tickets por Colas y estados, dejando registro de una Bitacora que se genera cada vez que se realice una acci√≥n dentro del sistema. El sistema, cuenta con un menu facil de navegar y utilizar. 
Los diferentes Tipos de Colas/Roles que se manejaran, son los siguientes: 

 - Mesa de Ayuda
 - Soporte Tecnico
 - Desarrollador

Siendo este ultimo el rol maximo y el final, en el cual recaeran las situaciones con un nivel alto de urgencia o complejidad.
